# 📖 Spell System

## Unlocking

- Learn spells by:
    - Reading books

    - Deciphering texts

- Starts a **Spell Unlock Quest**

## Structure

- Each element has **5 tiers**

- Each tier contains multiple spells

### Example:

- Tier 1 Fire

- Tier 2 Fire

- Tier 3 Fire

- Tier 4 Fire

- Tier 5 Fire
