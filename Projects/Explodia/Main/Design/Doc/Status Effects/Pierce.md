## Type
Offensive Status Effect

## Description
Allows attacks to bypass a portion of the target's defensive stats.

## Mechanics
- Ignores a percentage of Defense when calculating damage
- Only affects the current attack (not a persistent debuff)

## Variants
- Flat Pierce — ignores fixed defense value
- Percentage Pierce — ignores % of defense
- True Damage — ignores all defenses completely

## Interactions
- Does not reduce target stats (unlike [[Corrosive]])
- Works independently per hit
- Can stack with Critical Damage
